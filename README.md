# Real-world Data Project (Retail Sales)

## Objective
Analyze retail sales data to identify trends and business insights.

## Dataset
Custom retail sales dataset containing monthly sales and profit data.

## Tools Used
- Python
- Pandas
- Matplotlib
- GitHub

## Analysis Performed
- Statistical summary
- Category-wise sales analysis
- Data visualization

## Findings
- Electronics generated the highest sales.
- Clothing was the second-highest category.
- Retail sales increased toward the end of the year.
